# AYHEM Kernel Overview

النواة السيادية تربط كل وحدات المشروع تلقائيًا، بدون التعديل على الملفات القديمة.
- Visualization → عرض مرئي للنواة
- Bridge → ربط داخلي ووحدات قديمة
- Prediction → التنبؤ الذكي
- Voice → واجهة صوتية متقدمة
- Governance → الوثيقة العليا SUPREME_MANDATE.md
- History → سجل التشغيل الآلي
